# vROPs Archetype

vROps archetype used to pull and push content from/to vROps / VCF Operations.

## Overview

This archetype is used when you want to pull and push content from/to vROps / VCF Operations. The content you want to pull is defined in `content.yaml`. All exported content goes to `./src`.

Supported content:

- `view`
- `dashboard`
- `alert-definition`
- `symptom-definition`
- `policy`
- `default-policy`
- `recommendation`
- `super-metric`
- `metric-config`
- `report`
- `custom-group`

This Archetype is also known as `vrops`.
