# vRLI Archetype

vRLI archetype used to pull and push content from/to vRLI / VCF Operations for Logs.

## Overview

This archetype is used when you want to pull and push content from/to vRLI / VCF Operations for Logs. The content you want to pull is defined in `content.yaml`. All exported content goes to `./src`.

Supported content:

- `alerts`
- `content-packs`

This Archetype is also known as `vrli`.
